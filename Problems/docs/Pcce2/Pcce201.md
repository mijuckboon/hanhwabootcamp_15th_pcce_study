## 문제 정보
* 빈칸 채우기 문제

### 문제 코드
```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        String message = <빈칸>;

        System.out.println("3<빈칸>2<빈칸>1");
        System.out.println(message);
    }
}
```

### 제출 정답
```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        String message = "Let's go!"; // 채운 부분

        System.out.println("3\n2\n1"); // 채운 부분
        System.out.println(message);
    }
}
```
